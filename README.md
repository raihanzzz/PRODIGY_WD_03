# 🎮 Tic-Tac-Toe Pro Web App

A modern and interactive **Tic-Tac-Toe game** built using **HTML, CSS, and JavaScript** with advanced UI effects and smart gameplay logic.

This project transforms the classic game into a **dynamic web experience** with animations, AI opponent, and real-time score tracking.

---

## 🚀 Features

- 🎮 Player vs Player mode  
- 🤖 Player vs AI mode  
- 🧠 Smart AI decision-making  
- 📊 Live score tracking (X, O, Draws)  
- ✨ Smooth animations & transitions  
- 🎨 Modern glassmorphism UI  
- 🌌 Interactive animated background  
- 📱 Fully responsive design  

---

## 📂 Project Structure
```
│── index.html  
│── style.css  
│── script.js  
```
---

## 🧠 How It Works

- Maintains game state using an array  
- Checks winning conditions dynamically  
- Switches players after each move  
- AI logic:
  - Tries to win  
  - Blocks opponent  
  - Picks best available move  
- Updates UI in real-time  

---

## 🎮 Game Modes

### 👥 Player vs Player
- Two players play on the same device  

### 🤖 Player vs AI
- Play against a smart AI opponent  
- AI makes strategic moves for better gameplay  

---

## 🖥️ Interface

- 3x3 game board  
- Current player indicator  
- Scoreboard (X / O / Draws)  
- Status messages (win/draw/turn)  
- Control buttons (New Game, Reset Score)
